## Compile and run the project

For Backend
```bash
# into backend folder
$ cd backend

# setup project
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
For Frontend
```bash
# setup project
$ npm install
```
## Compile and run the project using Docker
```bash
# create and run
docker-compose up

# delete
docker-compose down
```
